## 综合测试脚本

```
curl -L https://gitlab.com/spiritysdx/za/-/raw/main/ecs.sh -o ecs.sh && chmod +x ecs.sh && bash ecs.sh
```

脚本会输出服务器的 CPU、内存、硬盘、网络速度、流媒体解锁能力和 IP 质量等详细信息。你可以重点关注 三网回程路由 和 流媒体解锁 部分
